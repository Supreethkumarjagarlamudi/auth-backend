1. Clone the repo:
   ```bash
   git clone https://github.com/Supreethkumarjagarlamudi/auth-backend.git
   cd auth-backend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Add a `.env` file:
   ```
   MONGO_URI=your_mongo_connection_string
   PORT=5000
   ```

4. Start the server:
   ```bash
   npm start
   ```
   Server runs at: `http://localhost:5000`
